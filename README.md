# 🏡 Real Estate Price Trend Analyzer

## 📌 Overview

This project analyzes real estate price trends using data science techniques and visualizations. It includes geo-spatial analysis to understand price distribution across locations.

---

## 🚀 Features

* 📊 Data Cleaning & Preprocessing
* 📈 Price Trend Analysis
* 🌍 Geo Visualization (Map-based insights)
* 📉 Statistical Analysis

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* Matplotlib / Seaborn
* Geopandas (for maps)

---

## 📂 Project Structure

* `Real Estate Price Trend Analyzer.ipynb` → Main notebook
* `dataset.csv` → Input dataset

---

## 📊 Key Insights

* Price variation across locations
* Trends over time
* High-value vs low-value regions

## 💡 Future Improvements

* Machine Learning price prediction
* Interactive dashboard deployment
* Real-time data integration
